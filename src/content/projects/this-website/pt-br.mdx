Este site é um experimento para manter a implementação proporcional ao conteúdo.

Ele é renderizado estaticamente, guarda os textos em MDX local, gera cada idioma durante o build e evita JavaScript no navegador para páginas comuns. O repositório é a fonte do conteúdo, o limite das traduções e a origem do deploy.
